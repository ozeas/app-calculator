# Anticipation Calculator App

Webapp de cálculo de atencipação de transações.

---

## Dependências internas

Possui como dependência interna o pacote @ac/components para _ui components_ e _theme_.

- @ac/components

## Desenvolvimento

```shell
cd app-calculator
yarn dev:app
```

### Testes

```shell
cd app-calculator
yarn test:app
```

### Build

```shell
cd app-calculator
yarn build:app
```
