# @ac/componentes UI components

@ac/components é uma base para iniciar, evoluir componentes customizados nas aplicações. Tra-se de uma lib que centraliza o desenvolvimento dos componentes de UI e configurações de tema.
Permitindo que as aplicações possam reutilizar e especificar componentes customizados a partir destes.

## Componentes

@ac/components oferece um conjunto inicial de componentes prontos para uso. Isso inclui componentes prontos para usar, como o UIInput, UIInputMask, e outros mais primitivos de baixo nível, como o UIText.

## Extensibilidade

O @ac/components é desenvolvimento utilizando as ferramentas `styled-components` e `styled-system`, permitindo facilmente <br />
extensão e personalização para componentes mais complexos.

Para mais detalhes veja a documentação do `styled-components` e `styled-system`.
